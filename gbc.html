Gameboy Color

Nintendo, 1999

The Game Boy Color (abbreviated as GBC) is an 8-bit handheld game console developed by Nintendo. It was released in Japan on October 21, 1998, and in international markets the following month. Compared with the original Game Boy, the Game Boy Color features a color TFT screen instead of monochrome, a CPU running at up to twice the speed, and four times as much memory. It is backward compatible with games developed for its predecessor.
